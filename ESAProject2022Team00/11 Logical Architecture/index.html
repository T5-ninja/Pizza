Display "Login is unsuccessful!"

Bring player to home page to begin the game
Reset game to initial state

Transits to factory tab
Chooses which ingredients to purchase using the slider

Updates cash position and record purchases from this factory

Update shop's inventory levels

Update pizza supply levels
